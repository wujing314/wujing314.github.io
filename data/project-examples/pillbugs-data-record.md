# PILL_BUGS 粗糙度数据采集实例

## 采集目标

对不同粗糙表面进行触角信号采集，为后续滤波、特征提取和分类模型准备数据。

## 表面类别

- 光滑亚克力板
- 细砂纸
- 粗砂纸
- 3D 打印纹理表面
- 软质橡胶表面

## 每次采集记录

| 字段 | 示例 |
| --- | --- |
| 日期 | 2026-06-xx |
| 表面类型 | 粗砂纸 |
| 触角材料 | TPU + 应变片 |
| 采样频率 | 1 kHz |
| 移动速度 | 低速 / 中速 |
| 放大电路 | 运放方案 A |
| 文件名 | rough_sandpaper_001.csv |

## 操作步骤

1. 固定触角安装角度和接触压力。
2. 让小车以固定速度通过测试表面。
3. 采集原始 ADC 数据，不要只保存处理后的结果。
4. 同时记录视频或图片，方便排查异常波形。
5. 每类表面至少采 20 组数据。

## 问题记录

- 如果波形几乎没有变化，优先检查触角是否真的产生形变。
- 如果噪声过大，检查接地、屏蔽、运放供电和采样线长度。
- 如果同类数据差异过大，检查速度和接触压力是否一致。
